# Supermarket

A full-stack e-commerce application built with **React**, **Node.js**, **Express**, and **PostgreSQL**. Users can browse products, manage a cart, place orders, and track deliveries. Admins can manage products, categories, orders, and users through a dedicated dashboard.

## Tech Stack

**Backend**
| Layer | Technology |
| ----- | ---------- |
| Runtime | Node.js |
| Framework | Express |
| Database | PostgreSQL (raw `pg`, no ORM) |
| Auth | bcrypt + jsonwebtoken |
| Validation | Joi |
| File uploads | Multer + Cloudinary |
| Config | dotenv |

**Frontend**
| Layer | Technology |
| ----- | ---------- |
| Framework | React 18 (Vite) |
| Styling | Tailwind CSS |
| Routing | React Router v6 |
| HTTP client | Axios |
| State | React Context (Auth + Cart) |
